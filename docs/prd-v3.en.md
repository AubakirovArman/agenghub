AgentHub v3 is the productization plan for the local-first CLI. The target milestone is AgentHub v0.2 — Installable Local Developer Preview.

The working backlog is in `prd/v3/`.

## Phases

1. Product rename audit.
2. Release engineering.
3. Installers and packaging.
4. Product CLI UX: doctor, version, providers, config.
5. Real provider gateway.
6. Product quality fixtures.
7. Security hardening.
